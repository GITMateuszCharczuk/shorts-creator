# Superpowers — specs & plans

This directory holds spec and plan artifacts produced while working with the
[superpowers](https://github.com/obra/superpowers-marketplace) Claude Code plugin
(installed automatically via the SessionStart hook in `.claude/hooks/session-start.sh`).

- **`specs/`** — specifications: the *what* and *why* of a feature/change before
  implementation (problem statement, requirements, acceptance criteria).
- **`plans/`** — implementation plans: the *how* (step-by-step approach, file changes,
  milestones) derived from a spec.

Keep one file per spec/plan, named with a short slug (e.g. `specs/voice-stage.md`,
`plans/voice-stage.md`).
